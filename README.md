# Projeto-Clone-Pinterest
Um clone do Pinterest feito utilizando Pyhton e HTML

Para iniciar o programa, execute o arquivo `main.py`

--Sumário dos Arquivos:

  - `criar_banco.py` - Cria o banco de dados (Já está criado, não precisa executar)
  - `instance` -> `comunidade.bd` - Banco de dados (Criado usando SQLAlchemy do Flask)
  - `__init__.py` - Configurações do sistema
  - `forms.py` - Formulários para coleta de dados
  - `models.py` - Modelo utilizado para a criação do banco de dados
  - `routes.py` - Rotas Flask utilizadas para carregar as páginas do sistema
  - `templates` - Pasta onde estão contidos os arquivos HTML
  - `static` - Pasta onde estão contidas: `css` - Pasta onde está contido o arquivo CSS; `fotos_post` - Pasta onde estão contidas as fotos enviadas pelos usuários; `fotos_site` - Pasta onde estão contidas as imagens       de fundo do sistema
